+ What is this plugin

This jQuery plugin was based on the Windows 8/8.1 User Interface (before: "Metro UI"), since there is some plugins and CSS for block' styles similar to Start UI Interface, but nothing about the modal alert.

There is some usage examples on the folder "DEMO", it includes how I made the MIT License' modal (the one you used for download this) and more.

After download it, upload the files on your server and add the minified version of the JS and CSS plugin on your page. Include those lines after the jQuery call line:

	
...				
	<script src="jmetro.min.js"></script>
	<link href="jmetro.min.css" rel="stylesheet" />
...

+ Just to start

For execute a simple alert modal, you just need to use that code, when the document is ready.

	
$.jMetro(
	"This is a title",
	"This is a message base, just a simple base."
);

It will show a default blue window alert, with title, message/details and a OK button that only will hide the modal.

... but you can set options for customize your modal, just adding an array of options:

	
$.jMetro(
	"... title ...",
	"... message ...",
	{
		style: "error"
	}
);

It will show an "ERROR" style (red window).
